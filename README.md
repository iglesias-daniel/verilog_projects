# verilog_projects
General exercises and projects that I have worked on using Verilog.

### Comandos útiles

Con los siguientes archivos:
- hello.v
- hello_tb.v

Puedes utilizar los siguientes comandos:

- iverilog -o hello_tb.vvp hello_tb.v
- vvp hello_tb.vvp

- gtkwave

Seleccionar: Open New Tab
